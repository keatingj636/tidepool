# 未来功能计划 / Future Feature Plans

## 🔄 recurring tasks / 周期性任务

**想法来源**: 2026-03-25

创建任务时增加"是否周期性"选项。

- **普通任务**: 加入某日计划后从任务池消失；关闭当日计划后归还池。
- **周期性任务** (如每日跑步): 加入计划后**仍留在任务池**，每天都可以快速添加，不需要重复创建。

**需要的后端改动**:
- `Task` 模型增加 `is_recurring: bool` 字段
- `GET /tasks/pool` 逻辑调整：周期性任务始终出现在池中，即使已有 DayPlanEntry

**前端改动**:
- 创建任务 modal 增加"周期性任务"开关
- 池中的周期性任务可以用不同样式标记（如循环图标）
